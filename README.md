# bot-dlya-tg

#### 1. Отчет по расписанию
Загружаете файл с расписанием группы
Бот считает количество пар по каждой дисциплине
Выводит итоговый отчет

#### 2. Отчет по темам занятий
Загружаете файл с темами уроков
Формат должен быть: `Урок № X. Тема: ...`
Бот выводит ошибки (неправильный формат)

#### 3. Отчет по студентам
Загружаете файл со студентами
Ищет студентов с:
редней оценкой за ДЗ = 1
  Оценкой за класс.работу < 3
Выводит список таких студентов

#### 4. Отчет по посещаемости
Загружаете файл посещаемости преподавателей
Ищет преподавателей с посещаемостью < 40%
Выводит отсортированный список

#### 5. Отчет по проверке ДЗ
Загружаете файл проверки домашних заданий
Ищет преподавателей с процентом проверки < 70%
  Поддерживает отчеты за месяц и неделю
Показывает: получено/проверено/процент

#### 6. Отчет по сдаче ДЗ

Загружаете файл сданных ДЗ студентами
Ищет студентов с процентом выполнения < 70%
Показывает: сдано/всего/процент

#### 7. ИИ поможник
Отвечает на реплайнутые сообщения с добавленном от пользователя контекстом
